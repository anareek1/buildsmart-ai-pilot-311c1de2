# СК-Казалем — ИИ-платформа управления стройкой

Цифровая платформа управления строительством для ТОО СК-Казалем.

## Stack

Vite + React + TypeScript + Tailwind + shadcn/ui

## Dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Production deployed at https://sk-kazalem.kz
